# 后台功能总览与缺口清单

本文档记录当前后台已经具备的能力，以及对照早期博客后台规划后仍缺少的功能。它用于后续开发排期，不作为前台展示页面。

## 当前已实现

### 账号与初始化

- 后台登录：支持管理员用户名、密码登录。
- 会话管理：使用 Cookie Session 保存后台登录状态。
- 退出登录：后台顶部支持退出当前会话。
- 首次部署向导：当数据库或管理员密码未配置时，引导填写 PostgreSQL 连接信息和管理员账号。
- 配置来源识别：可区分数据库和认证配置来自环境变量、运行时文件或缺失状态。

### 仪表盘

- 文章统计：展示文章总数、已发布数量、草稿数量。
- 扩展统计：展示分类数量、评论数量、待审核评论数量、内容图片数量和图片总占用。
- 快捷操作：提供新建文章、内容图片、站点设置、分类、评论、功能管理入口。
- 版本信息：展示当前后台版本。
- 帮助信息：展示后台功能文档位置和运行时配置存储提示。
- 发布趋势：按最近月份展示文章数量趋势。
- 最近动态：展示最近更新的文章状态与更新时间。
- 最近文章：提供进入文章管理页和编辑文章的快捷入口。
- 数据刷新：支持手动刷新仪表盘数据。
- 空配置提示：数据库未配置时显示说明，不阻断后台页面渲染。

### 文章管理

- 文章列表：展示标题、slug、发布状态、创建时间、更新时间。
- 搜索文章：支持按标题、slug、摘要搜索。
- 状态筛选：支持全部、已发布、草稿筛选。
- 分页浏览：列表支持上一页、下一页分页。
- 新建文章：支持创建数据库文章。
- 编辑文章：支持修改标题、slug、摘要、正文、封面图和发布状态。
- 删除文章：支持从 PostgreSQL 删除已有文章。
- 预览文章：已发布文章打开正式地址，草稿文章通过 preview 参数查看。
- Markdown 编辑：正文使用 Markdown 编写。
- Markdown 预览：桌面端左右编辑预览，移动端可切换编辑和预览。
- 封面设置：支持封面图 URL。
- 封面布局：支持右图左文、左图右文、上半屏背景三种布局。
- 发布控制：支持发布到前台或保存为草稿。

### 内容图片

- 图片上传：支持 JPG、PNG、WEBP、GIF。
- 图片列表：按更新时间展示已上传图片。
- 图片预览：以网格卡片展示缩略预览。
- 复制地址：支持复制图片 URL，用于文章正文或封面图。
- 删除图片：支持删除已上传图片。
- 存储路径：图片存放在 `public/uploads/content-images`，访问路径为 `/uploads/content-images/...`。
- 基础统计：展示图片数量、总占用和存放路径。

### 系统设置

- 系统信息页：展示数据库是否配置、数据库类型、认证配置状态、默认用户名、会话有效期、Session Secret 状态。
- 只读技术信息：展示当前编辑器方案、ORM 方案和后台 UI 栈。
- 站点全局设置：支持维护网站名称、Logo、备案号、版权信息、后台地址、网站开启状态和闭站提示。
- SEO 设置：支持维护首页 title、description、keywords。
- 导航设置：支持维护前台导航标题、路径、内链或外链、上级导航、排序和启用状态。
- 内容设置：支持保存远程图片抓取、外链过滤、图片压缩、图片尺寸、缩略图尺寸、缩略图模式和默认缩略图配置。
- 账号设置：支持在后台修改管理员用户名和密码；当密码来自环境变量时不允许在后台改写。
- 设置保存：站点、SEO、内容和功能基础配置写入 PostgreSQL `admin_settings` 表。
- 独立数据表：导航写入 `admin_navigation`，分类写入 `admin_categories`，评论写入 `admin_comments`，友情链接写入 `admin_friend_links`。
- 旧数据迁移：首次读取后台功能配置时，会尝试将 `.data/admin-features.json` 自动导入 PostgreSQL。
- 账号配置：管理员账号运行时配置继续写入 `.data/runtime-setup.json`；当密码来自环境变量时不允许在后台改写。

### 分类管理

- 分类列表：展示当前维护的文章分类。
- 新增分类：支持添加分类名称、slug、排序和说明。
- 编辑分类：支持修改分类名称、slug、上级分类、排序和说明。
- 多级分类：支持选择上级分类。
- 删除分类：支持删除分类，并清理子分类的上级引用。

### 评论管理

- 评论列表：展示作者、邮箱、文章 slug、内容和审核状态。
- 状态筛选：支持全部、待审核、已通过、垃圾评论筛选。
- 审核处理：支持修改评论状态。
- 删除评论：支持删除评论记录。
- 示例录入：支持手动新增示例评论，便于后台审核流程测试。

### 功能管理

- Robots 管理：支持维护 robots.txt 文本规则。
- Sitemap 管理：支持启用或关闭 Sitemap，并选择 TXT 或 XML 格式。
- 友情链接：支持新增、编辑、排序、启停和删除友情链接。
- 搜索引擎推送：支持保存百度推送 Token、必应推送 Token 和 JS 推送代码。

### 后台框架

- Admin 布局：后台使用独立布局，不显示前台导航和页脚。
- 侧边栏导航：包含仪表盘、文章、内容图片、站点设置、分类、评论、功能管理、系统设置入口。
- 侧边栏折叠：桌面端支持收起和展开侧边栏。
- 移动端抽屉：小屏幕通过顶部菜单打开侧边栏。
- 顶部搜索：文章列表页顶部支持搜索输入。
- 前台入口：顶部提供返回前台入口。
- 暗色模式：后台沿用站点暗色模式切换。

## 与原规划相比缺少的功能

### 系统设置

- 前台接入：站点名称、Logo、备案、版权、SEO、导航配置已可保存，前台页面尚未统一读取这些配置。
- 闭站拦截：网站状态和闭站提示已可保存，尚未接入前台访问拦截。
- 多管理员：账号设置目前只支持单管理员修改用户名和密码，缺少多管理员、角色权限和操作日志。

### 内容设置

- 远程图片抓取执行：配置已可保存，缺少将正文远程图片自动保存到本地的处理流程。
- 外链过滤执行：配置已可保存，缺少文章正文外链过滤或 rel 策略处理。
- 图片压缩执行：配置已可保存，缺少上传图片自动压缩处理。
- 指定尺寸缩放执行：配置已可保存，缺少统一宽度、高度或最大边处理。
- 缩略图生成执行：配置已可保存，缺少缩略图文件生成流程。
- 批量处理：缺少批量重新生成缩略图、批量压缩历史图片。

### 内容管理

- 文章分类关联：分类可维护，文章编辑页尚未选择或保存分类。
- 评论提交入口：评论后台可维护，前台尚未提供评论提交入口。
- 评论回复：评论管理缺少后台回复能力。
- 评论审核策略：缺少默认审核开关、敏感词、黑名单等策略。
- 图片替换：内容图片目前可上传和删除，缺少替换文件能力。
- 图片批量操作：缺少批量删除、批量复制、按类型或日期筛选。
- 文章批量操作：缺少批量发布、批量转草稿、批量删除。

### 功能管理

- Robots 发布：规则已可保存，缺少公开 `/robots.txt` 输出。
- Sitemap 生成：配置已可保存，缺少 sitemap 生成、刷新、下载和公开访问。
- 友情链接前台展示：友链已可维护，前台尚未读取展示。
- 搜索引擎主动推送：Token 和脚本已可保存，缺少实际推送执行记录。

## 建议优先级

### P0

- 后台文档化：保持本文档随功能变化同步更新。
- 内容图片与文章编辑器联动：在文章编辑页直接选择已上传图片，自动插入 Markdown 图片或封面 URL。
- 前台接入站点配置：让首页 SEO、导航、版权、备案、站点状态读取后台配置。
- 文章分类关联：文章编辑页支持选择分类，文章列表支持分类筛选。

### P1

- 评论提交入口：前台文章详情接入评论提交，后台审核后展示。
- Robots 和 Sitemap 输出：生成公开 SEO 文件，并支持刷新。
- 图片处理流水线：接入压缩、缩放、缩略图和默认缩略图处理。
- 评论审核策略：支持默认审核开关、敏感词和黑名单。

### P2

- 多管理员与权限：支持多账号、角色、权限分配和操作日志。
- 批量操作：支持文章批量发布、转草稿、删除和图片批量处理。
- 搜索推送执行：支持百度、必应主动推送和执行结果记录。
- 友情链接展示：前台接入友链列表。

## 维护说明

- 新增后台页面、API 或数据模型时，应同步更新“当前已实现”。
- 开始实现缺口功能时，应从“建议优先级”移动到具体开发计划。
- 功能完成后，应从“缺少的功能”移入“当前已实现”。
